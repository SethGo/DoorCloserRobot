# Door Closer Robot

The .ino code file is in the DoorCloser folder. Ready to upload to Arduino.

# Components:
- Straight Link Piece, 7cm
- Shaft Coupler
- NEMA 17 Motor with 51:1 Gearbox
- IR Obstacle Sensor
- Arduino Uno
- 9v Battery Pack
- AVAWO DC 24V Power Supply
- DM542T Stepper Motor Driver
